Fork / customize the OpenClaw Control UI (Vite + Lit)
=====================================================

Layout in this repo:

  control-ui/     — Vite + Lit app (edit here for UI). Must keep control-ui/tsconfig.json
                    (experimentalDecorators + useDefineForClassFields: false) or the production
                    bundle can break with SyntaxError in Electron/Chromium.
  openclaw-src/   — Shared TypeScript from upstream OpenClaw (gateway types, helpers).
                    The UI imports this via the "@openclaw" alias; keep it in sync when you
                    merge upstream, or trim only if you know the dependency graph.

Prerequisites: Node 22.16+ (OpenClaw engines).

From the desktop repo root:

  npm run control-ui:install
  npm run control-ui:build     → output: dist/control-ui/
  npm run control-ui:dev       → Vite on :5173 (see OpenClaw docs for gatewayUrl query)

The desktop app sets gateway.controlUi.root when it finds:

  - vendor/control-ui/index.html   (optional: npm run control-ui:vendor after build), or
  - dist/control-ui/index.html

Disable custom UI: OPENCLAW_DESKTOP_SKIP_CUSTOM_CONTROL_UI=1

Custom root: OPENCLAW_DESKTOP_CONTROL_UI_ROOT=<absolute path to folder with index.html>

Packaging: after control-ui:vendor, add electron-builder extraResources for vendor/control-ui
if you need the fork inside the installer.
