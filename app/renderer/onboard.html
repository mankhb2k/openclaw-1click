Chào mừng đến OpenClaw Desktop

Chọn cách dùng AI cho lần đầu. Bạn có thể dùng DeepSeek-R1 cục bộ qua Ollama (không cần API cloud), hoặc cấu hình API nhà cung cấp. Bạn cũng có thể bỏ qua và cấu hình sau trong Dashboard.
